# Bangla font
Embed SutonnyMJ to web pages and apps, the easiest way.

## How To Use
Simple add the following line to HTML.

``` <link href='https://saidylive.github.io/fonts/stylesheet.css' rel='stylesheet' type='text/css'> ```
And then set the following CSS to the target element.

```
.container {
    font-family: 'sutonnymjregular';
    font-family: 'sutonnymjbold';
    font-family: 'sutonnymjbold_italic';
    font-family: 'sutonnymjitalic';
    font-family: 'sutonnyomjregular';
}
```
